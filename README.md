# Personal Portfolio Website

## Project Overview

This project is a personal portfolio website developed using HTML5 as
part of a Web Development Internship. The objective of the project is to
create a simple and well-structured webpage that presents personal
information, technical skills, projects, and a contact form.

## Features

-   About Me section
-   Skills section
-   Projects section
-   Contact form
-   Internal navigation links
-   Profile image with alternative text
-   HTML form validation
-   Semantic HTML5 structure

## HTML Concepts Learned

Through this project, I learned the basic structure of an HTML document
and the use of HTML tags and elements. I used headings, paragraphs,
lists, links, images, forms, text areas, and buttons.

I also learned how HTML attributes such as `id`, `href`, `src`, `alt`,
and `required` are used. Semantic HTML elements such as `header`, `nav`,
`main`, `section`, and `footer` were used to organize the webpage
clearly.

## Setup Instructions

1.  Download or clone the project repository.
2.  Open the project folder in Visual Studio Code.
3.  Open `index.html` in a web browser or run it using Live Server.
4.  Use the navigation links to move between the different sections of
    the portfolio.

## Code Structure

The `index.html` file contains the complete HTML5 structure of the
portfolio website.

-   The `header` contains the name, career title, and navigation menu.
-   The `main` element contains the About, Skills, Projects, and Contact
    sections.
-   The About section contains profile information and an image.
-   The Skills section uses an HTML list.
-   The Projects section presents project information using headings and
    paragraphs.
-   The Contact section contains a form with name, email, and message
    fields.
-   The `footer` contains copyright information.

## Technical Details

The project uses HTML5 semantic elements to provide a clear and
meaningful webpage structure. Internal navigation is implemented using
anchor links connected to section IDs.

The contact form uses `label`, `input`, `textarea`, and `button`
elements. The `required` attribute provides basic required-field
validation, and the email field uses `type="email"` for browser-based
email format validation.

The profile image uses the `alt` attribute to provide alternative text.
